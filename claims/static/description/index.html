Warranty Module

The Warranty module allows the user to claim a request based on the products warranty specified in months. .

Create a claim request

The Claim Menu is placed in the Sales Main Menu.

The created claims are categorised in the tree view based on their priority and the current state of the claim.

Create Claim line Page

The user has to select the customer and all the invoices associated with that customer will be listed in the drop down menu.

On selecting the invoice number all the products related to that invoice will be listed below, the user can delete the products from the available list. If the product is in warranty the record will be displayed in green color, if the warranty of the product has expired the record will be turned into red color.

The user can change the state of the claim using the buttons listed in the header. By default the state changes to 'Pending' on saving he current entry.

Final claims page

Only the claims with 'Approved' state will be displayed in final claims.
